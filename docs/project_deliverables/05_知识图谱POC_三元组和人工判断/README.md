# 知识图谱 POC：三元组和人工判断

这里对应之前用 Label Studio JSON 做的知识图谱小实验。

## 这条线做了什么

流程是：

`JSON → 文本块 → chunk → schema → 三元组 → evidence → 人工判断 → 知识图谱图片`

这个 POC 的价值不是证明最终 GraphRAG 已经完成，而是证明你已经理解并跑过“知识图谱构建”的基本链路。

## 这里的文件

- `知识图谱图片.svg`：可以直接展示的知识图谱图片。
- `三元组审核页面.html`：查看三元组和 evidence 的页面。
- `人工判断小结.md`：你人工判断后的结果摘要。
- `POC流程报告.md`：这一轮 POC 的流程说明。
- `schema_实体和关系规则.json`：实体类型、关系类型和证据字段规则。
