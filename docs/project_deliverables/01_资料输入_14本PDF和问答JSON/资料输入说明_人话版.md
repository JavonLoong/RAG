# 资料输入说明

这一步解决的是：老师给了一批资料，我们先判断哪些资料能直接读，哪些必须 OCR。

## 当前资料类型

- PDF 资料：14 本。
- 扫描版 PDF：13 本，需要 OCR。
- 可直接抽文本 PDF：1 本，可以直接进入普通 RAG。
- 问答 JSON：1 个，作为已有结构化文本进入普通 RAG。

## 为什么要先判断

因为普通 RAG 不是直接“看 PDF 图片”。如果 PDF 里面本质是扫描图片，就必须先 OCR；如果 PDF 本身有文字层，就可以直接抽文本。

## 汇报时怎么说

可以说：

> 先对老师给的资料做了可抽取性判断，把扫描版和可直接抽文本的资料分开处理。
