兜底与边界 Fallback and Boundaries

演示失败
不现场排环境。立即打开 offline fallback evidence：截图、browser smoke report、KG artifacts、readiness gate。
不过度承诺
主动声明 no award guarantee。Readiness 证明包完整，不证明特等奖概率。
硬证据边界
真实专家反馈和真实计时彩排归档前，不宣称 real expert feedback 或 real timed rehearsal 已完成。
