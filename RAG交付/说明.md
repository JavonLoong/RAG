## 本次更新内容

### 1. Label Studio JSON 入库 — 已支持
后端 parsing.py 已有完整的 Label Studio JSON 解析器，上传 JSON 后自动解析标注文本入库。
清华云盘下载的 27 个 JSON 是增量导出，只需上传最新最大的那一个即可。

### 2. 导出 ChromaDB — 已实现
- 「系统总览」页面：集合管理面板可导出整库 ZIP 或单集合 JSON
- 「数据接入」页面：新增「导出与备份」面板

### 3. 集合选择（新建/已有 DB 提示）— 已实现
点击「处理选中文件」后弹出集合选择对话框，可选已有集合或输入新名称创建。

## 文件清单
```
frontend_app/
  index.html          # 前端控制台（含全部 UI 逻辑）

api_server/
  server.py           # 后端入口（启动：python server.py）
  api.py              # FastAPI 路由（上传/处理/检索/导出/压测）
  parsing.py          # 文件解析器（支持 PDF/JSON/DOCX/TXT/MD/CSV 等）
```
