流程拓扑

系统结构保留为白底、高密度、可复核的工业流程视图。

从上传、解析、清洗、分块、向量化到检索与压测，每个阶段都保持可见和可验证。

01 上传与预检

支持单文件、批量文件与目录拖拽；前端在上传前过滤扩展名，后端再次验证类型并保留相对路径。

02 解析与清洗

JSON 支持更复杂结构提取，PDF / DOCX / TXT / Markdown / CSV / TSV / LOG 统一走解析分派，并汇总质量问题。

03 分块与向量化

标题触发分块和多级断点切分同时保留，chunk metadata 额外记录来源扩展名与来源类型。

04 检索与基准

语义检索、延迟统计与本地压测集成在同一控制台中，便于做链路回归和性能观察。
