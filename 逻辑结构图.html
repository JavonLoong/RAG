五、各模块核心职责速查

层 模块 核心职责 关键类 / 函数
入口 __main__.py CLI 入口：serve / ingest / search / stats / benchmark / demo main()
入口 api.py FastAPI REST API，前端静态文件挂载，CORS create_app()
编排 pipeline.py 入库 & 查询全流程编排，ChromaDB 客户端管理与资源释放 ingest_source_payloads(), query_collection()
处理 parsing.py 多格式解析：JSON (含 Label Studio) / PDF / DOCX / TXT / MD / CSV / TSV load_source_payload()
处理 cleaning.py OCR 碎片修复，短文本合并 (min_chars=10) clean_records()
处理 chunking.py Title 触发分块 + 多级断点 + overlap 滑动窗口 chunk_records(), split_text_with_overlap()
处理 embeddings.py BGE-m3 向量嵌入，不可用时自动降级 Hashing create_embedding_backend()
检索 retrieval.py 语义(BGE) + BM25(jieba) + RRF 混合检索融合排序 HybridRetriever, search_hybrid()
检索 rag.py 检索 → 拼接上下文 → 模板回答（预留 LLM API 接口） rag_answer(), build_llm_prompt()
辅助 schemas.py 数据类定义：SourceRecord, TextBlock, ChunkRecord @dataclass
辅助 text_utils.py 文本规范化、稳定哈希、token 估算 normalize_text(), stable_hash()
辅助 benchmark.py 合成数据性能基准测试 run_synthetic_benchmark()
